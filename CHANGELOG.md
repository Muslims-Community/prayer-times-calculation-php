# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-10-15

### Added
- Initial release of Prayer Times Calculation PHP package
- Core PrayerTimesSDK class for prayer times calculation
- Support for multiple calculation methods (MWL, ISNA, Egypt, Makkah, Karachi, Custom)
- Support for different Asr jurisdictions (Standard, Hanafi)
- Laravel service provider and facade integration
- Configuration file for Laravel projects
- Comprehensive test suite with PHPUnit
- Astronomical utility functions
- Framework-agnostic design
- Complete documentation and examples

### Features
- Offline prayer times calculation
- Accurate astronomical algorithms
- PHP 8.0+ support with modern PHP features (enums, typed properties)
- Laravel auto-discovery support
- Configurable calculation methods and angles
- Time formatting and manipulation utilities
